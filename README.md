# 12-Employee-Tracker

## Description
Code for an employee tracking system using MySql for the backend database

## Usage

To use it invoke the following command in the terminal:

```bash
nmp i
```

Followed by:
```bash
mysql.server start
```

This application is deployed on GitHub pages.
